## 一、六大意图类型模板

### 1. 数值查询 (number) - 目标占比 35%

```
- {公司}2024年营业收入是多少？
- {公司}2024年归母净利润是多少亿元？
- {公司}2024年毛利率是多少？
- {公司}2024年资产负债率是多少？
- {公司}2024年ROE（净资产收益率）是多少？
- {公司}2024年每股收益（EPS）是多少？
- {公司}2024年经营活动产生的现金流量净额是多少？
- {公司}2024年研发费用占营收比例是多少？
- {公司}2024年末总资产规模是多少？
- {公司}2024年末员工总数是多少？
```

### 2. 事实查询 (factual) - 目标占比 20%

```
- {公司}的主营业务是什么？
- {公司}2024年的董事长是谁？
- {公司}2024年的审计机构是哪家？
- {公司}的总部在哪里？
- {公司}2024年主要产品有哪些？
- {公司}2024年有哪些重要子公司？
- {公司}的股票代码是什么，在哪个交易所上市？
- {公司}2024年分红方案是什么？
```

### 3. 布尔判断 (boolean) - 目标占比 15%

```
- {公司}2024年营收是否同比增长？
- {公司}2024年净利润是否为正？
- {公司}2024年是否进行了现金分红？
- {公司}2024年资产负债率是否超过70%？
- {公司}2024年是否存在商誉减值？
- {公司}2024年研发投入是否超过营收的5%？
- {公司}2024年经营性现金流是否为正？
- {公司}2024年是否有海外业务收入？
```

### 4. 对比分析 (comparative) - 目标占比 15%

```
- {公司}2024年与2023年的营业收入相比变化了多少？
- {公司}2024年的毛利率与上年相比有何变化？
- {公司}的净利润率与行业平均水平相比如何？
- {公司}2024年各业务板块收入占比分别是多少？
- {公司}国内与海外收入的比例是多少，变化趋势如何？
- {公司}2024年前五大客户销售额占比与上年相比有何变化？
```

### 5. 开放性分析 (open) - 目标占比 10%

```
- {公司}2024年面临的主要风险因素有哪些？
- {公司}2024年的核心竞争优势是什么？
- {公司}未来发展战略是什么？
- {公司}2024年的行业发展趋势分析是怎样的？
- 请分析{公司}2024年的盈利质量。
- {公司}2024年在ESG方面有哪些重要举措？
```

### 6. 名称/实体查询 (name) - 目标占比 5%

```
- {公司}2024年的审计意见类型是什么？
- {公司}的证券代码和简称是什么？
- {公司}的注册地址是哪里？
- {公司}2024年新增了哪些专利？
```

## 二、三级难度标准

### Easy (简单) - 约 40%
- 答案为单一数值或事实，可从表格/摘要中直接获取
- 示例: "贵州茅台2024年营业收入是多少？"
- 预期: 单个精确数字 + 出处页码

### Medium (中等) - 约 40%
- 需要跨段落/跨表格综合，或需简单计算/比较
- 示例: "比亚迪2024年新能源汽车业务与传统燃油车业务收入占比分别是多少？"
- 预期: 多个数字 + 比较分析

### Hard (困难) - 约 20%
- 需要跨多页推理、数值计算、多维度综合判断
- 示例: "请分析宁德时代2024年的盈利质量（经营现金流/净利润比值、应收账款周转率变化）"
- 预期: 多步推理 + 数据支撑 + 分析结论

## 三、行业专属难题模板 (Hard)

### 银行业 (招商银行)
```
- 招商银行2024年净息差是多少？与上年相比变化了多少个基点？
- 招商银行2024年不良贷款率和拨备覆盖率分别是多少？资产质量是否改善？
- 招商银行2024年零售业务利润占比是多少？零售客户AUM总量是多少？
```

### 保险/综合金融 (中国平安)
```
- 中国平安2024年个险新业务价值是多少？同比变化如何？
- 中国平安2024年寿险及健康险业务的综合偿付能力充足率是多少？
- 中国平安科技板块2024年营收和归母净利润各是多少？
```

### 新能源车 (比亚迪, 宁德时代)
```
- 比亚迪2024年新能源汽车销量是多少万辆？其中纯电动和插混各占多少？
- 宁德时代2024年储能业务营收同比增速是多少？占总营收比重变化如何？
- 宁德时代2024年电池出货量中海外占比是多少？
```

### 半导体 (北方华创, 寒武纪)
```
- 北方华创2024年在手订单金额是多少？同比增长多少？
- 寒武纪2024年的研发费用是多少？研发人员占总员工比例是多少？
- 寒武纪2024年是否实现盈利？亏损收窄了多少？
```

### 医药 (迈瑞医疗)
```
- 迈瑞医疗2024年海外收入占比是多少？同比变化如何？
- 迈瑞医疗2024年三大业务线（生命信息与支持、体外诊断、医学影像）营收分别是多少？
```

### 能源 (中国石油, 长江电力)
```
- 中国石油2024年原油产量和天然气产量分别是多少？
- 长江电力2024年总发电量是多少亿千瓦时？六座梯级电站各贡献多少？
```

### 白酒 (贵州茅台)
```
- 贵州茅台2024年茅台酒和系列酒销售收入分别是多少？占比变化如何？
- 贵州茅台2024年直销渠道收入占比是多少？i茅台平台销售额是多少？
```

### 资源材料 (紫金矿业, 海螺水泥)
```
- 紫金矿业2024年矿产金、矿产铜、矿产锌的产量分别是多少？
- 海螺水泥2024年水泥和熟料销量分别是多少？吨均售价变化如何？
```

### 消费电子 (立讯精密)
```
- 立讯精密2024年消费电子业务与通讯互联业务收入占比分别是多少？
- 立讯精密2024年前五大客户销售额占总营收的比例是多少？
```

### 券商 (中信证券)
```
- 中信证券2024年经纪业务、投行业务、资管业务收入分别是多少？
- 中信证券2024年末客户资产管理规模是多少？同比变化如何？
```

## 四、跨文档/边界测试题模板

这些题目用于测试系统在边界条件下的表现，需手动添加。

### 否定/陷阱题
```
- {公司}2024年是否发生了退市风险警示？（预期答案: 否）
- {公司}2024年有没有变更会计师事务所？
- {公司}2024年是否出现过重大安全事故？
```

### 跨段落推理
```
- 根据{公司}2024年报，计算其流动比率（流动资产/流动负债）。
- 根据{公司}2024年报，计算营收增速和净利润增速的差异说明了什么？
```

### 不存在信息
```
- {公司}2025年第一季度营收是多少？（预期: 系统应回答无法找到该信息）
- {公司}的量子计算业务营收是多少？（预期: 系统应回答该公司无此业务）
```

## 五、使用说明

### 生成流程
1. 在 `/rag/test-sets` 创建测试集
2. 对每家已同步公司，使用 LLM 自动生成 15-20 题
3. 对照本模板检查题型覆盖和难度分布
4. 手动补充 Hard 难度和边界测试题
5. 为每题标注 `reference_pages`（参考页码）
6. 使用 `expandQuestion()` 生成每题 3 个变体

### 验收标准
- P0 阶段: 50 题 (3-5 家公司，每家 10-15 题)
- P1 阶段: 100 题 (覆盖全部六种意图类型 + 三个难度等级)
- P2 阶段: 200 题 (覆盖全部 14 家公司 + 行业专属题)

### 评分基线目标
- Recall@10 >= 0.85
- NDCG@10 >= 0.70
- Accuracy >= 0.80
- Latency < 500ms
